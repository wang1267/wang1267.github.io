00_课题要求

课题要求

📋 任务目标

小车从始发线出发，沿黑色轨迹线自动循迹行驶，依次经过停止线1和停止线2，并在指定位置完成停车。

阶段 起始位置 目标位置 行驶状态
第一阶段 始发线 → 停止线1 正常行驶
第二阶段 停止线1 → 停止线2 减速行驶
第三阶段 停止线2 → 终点 停车
🧭

循迹逻辑

4 路红外传感器横向排布，检测黑白路面反射差异。中间两路对准黑线两侧，外侧两路辅助判断大角度偏离。

左传感器见黑 → 左转纠偏
右传感器见黑 → 右转纠偏

🅿

停车逻辑

红外传感器对横向黑线（停止线）进行计数。每次全部传感器同时检测到黑色横线时，计数器+1。

第1次见横线 → 正常行驶
第2次见横线 → 停止线1停车
第3次见横线 → 停止线2停车

⚙️

运动控制

位置式 PID 控制器根据传感器偏差实时计算左右轮差速，PWM 调节电机转速，实现平滑循迹。

Kp·误差 + Ki·积分 + Kd·微分
→ 左/右轮 PWM 差速输出

🔴 红外传感器 D0 输出逻辑

白色地面 → 反射红外 → 接收管收到 → D0 = 1（高电平，指示灯亮）
黑色轨迹 → 吸收红外 → 接收管未收到 → D0 = 0（低电平，指示灯灭）

⚡ L298N 电机控制逻辑

IN1=1 IN2=0 → 左电机正转
IN1=0 IN2=1 → 左电机反转
IN1=IN2 → 刹车/停转
ENA/ENB → PWM 占空比调速
